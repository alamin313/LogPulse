What LogPulse Does 

Multiple applications send logs and metrics to LogPulse.
LogPulse:

Ingests logs at high throughput

Streams them via Kafka

Stores raw logs in MongoDB

Stores configs & metadata in PostgreSQL

Caches hot data in Redis

Runs ML to detect anomalies

Generates alerts

Serves dashboards & queries via REST APIs
